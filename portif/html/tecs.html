Minhas Tecnologias:

/Python

Python é uma linguagem de programação versátil, conhecida por sua simplicidade e legibilidade. Ela é usada para desenvolver uma ampla gama de aplicativos, desde web e desktop até análise de dados e inteligência artificial.

Pros

+ Simplicidade e legibilidade do código.

+ Grande comunidade de desenvolvedores.

+ Versatilidade em diferentes domínios, como web, análise de dados e aprendizado de máquina.

Contras

- Desempenho relativo em comparação com linguagens compiladas.

- Consumo de recursos computacionais.

- Limitações em desenvolvimento mobile em comparação com outras linguagens estabelecidas.

Projetos

• YouTube: plataforma de compartilhamento de vídeos.

• Instagram: rede social de compartilhamento de fotos e vídeos.

• Dropbox: serviço de armazenamento em nuvem.

• Spotify: serviço de streaming de música.

• Netflix: plataforma de streaming de vídeo.

•NASA: agência espacial que utiliza Python em várias aplicações.

/HTML5

HTML (HyperText Markup Language) é a linguagem padrão para a criação de páginas da web, definindo a estrutura e o layout do conteúdo, como texto, imagens e links.

Pros

+ Fácil aprendizado.

+ Ampla compatibilidade com navegadores.

+ Flexibilidade na criação de layouts e estruturas personalizadas.

Contras

- Limitações de design, requerendo o uso de CSS.

- Complexidade em projetos grandes.

- Possíveis problemas de compatibilidade entre navegadores.

Projetos

• Mercado Livre

• Twitter

• G1

• YouTube

• Moodle

• Gov

/CSS3

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência e o layout de elementos HTML em uma página da web. Ele controla aspectos como cor, fonte, posicionamento e animação, permitindo a personalização visual dos elementos.

Pros

+ Separação de preocupações.

+ Controle detalhado da aparência.

+ Reutilização de estilos.

Contras

- Curva de aprendizado inicial.

- Incompatibilidade entre navegadores.

- Limitações de layout.

Projetos

• Mercado Livre

• Twitter

• G1

• YouTube

• Moodle

• Gov

/JavaScript

JavaScript é uma linguagem de programação de alto nível e orientada a objetos. É amplamente usada para criar interatividade em páginas da web, permitindo que os elementos da página sejam manipulados dinamicamente e respondam a eventos do usuário.

Pros

+ Interatividade.

+ Versatilidade.

+ Grande ecossistema e bibliotecas.

Contras

- Incompatibilidade entre navegadores.

- Segurança.

- Desempenho.

Projetos

• Amazon.

• Instagram.

• Twitch.

• Frameworks (React e Angular).

• Jogos online.

• OpenAI.

/SQL

SQL (Structured Query Language) é uma linguagem de programação usada para gerenciar e manipular bancos de dados relacionais. Com o SQL, é possível realizar operações como inserir, atualizar, recuperar e excluir dados de forma eficiente.

Pros

+ Gerenciamento de dados estruturados.

+ Alta flexibilidade.

+ Padronização.

Contras

- Complexidade.

- Desempenho dependente da estrutura do banco de dados.

- Limitações em relação a outros tipos de dados.

Projetos

• Sistemas de gerenciamento de bancos de dados - SGBDs (MySQL, PostgreSQL, Oracle e Microsoft SQL Server).

• Aplicações empresariais

• Sistemas de gestão de conteúdo (CMS) que armazenam e gerenciam conteúdo de websites.

• Aplicações de análise de dados que exigem consultas complexas.

• Websites e aplicativos que dependem de banco de dados para armazenar informações

• OpenAI

/Kali Linux

Kali Linux é uma distribuição de Linux baseada no Debian, desenvolvida especificamente para testes de segurança e auditoria de sistemas. Ele é projetado para ser uma poderosa plataforma de teste de penetração, fornecendo uma ampla gama de ferramentas de hacking ético e segurança cibernética.

Pros

+ Ferramentas especializadas.

+ Comunidade ativa.

+ Foco na segurança.

Contras

- Complexidade.

- Uso inadequado.

- Incompatibilidade com hardware específico.

Projetos

• Testes de segurança e penetração de sistemas e redes.

• Auditoria de segurança e conformidade em empresas e organizações.

• Pesquisa em segurança cibernética e desenvolvimento de técnicas de proteção.

• Treinamento e certificações em segurança cibernética.

• Atividades de resposta a incidentes e investigação forense digital.

• Realização de testes de segurança em redes Wi-Fi.

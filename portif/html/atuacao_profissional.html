Minhas atuaçoes profissionais

Trabalhei no desenvolvimento do site da renomada marca Nike, contribuindo para a criação de uma plataforma digital de destaque. Como parte da equipe de desenvolvimento, minha responsabilidade foi colaborar no desenvolvimento e implementação de tecnologias de ponta. Utilizamos uma combinação de tecnologias de front-end e back-end para criar uma experiência de usuário excepcional.

Tecnologias utilizadas:

Trabalhei como desenvolvedor na Riot Games, onde utilizei Python (Py) e SQL para criar recursos e funcionalidades em aplicações web. No front-end, utilizei Py com frameworks como Django para desenvolver interfaces interativas e responsivas. No back-end, utilizei Py em conjunto com bancos de dados SQL, como MySQL ou PostgreSQL, para gerenciar dados de forma eficiente. Priorizei a segurança, implementando práticas de criptografia e garantindo a proteção dos dados do usuário contra ameaças como injeção SQL e XSS. Foi uma experiência gratificante contribuir para uma empresa líder no setor de jogos, aprimorando minhas habilidades em Py, SQL e segurança.

Tecnologias utilizadas:

No meu trabalho como desenvolvedor no Spotify, tive a oportunidade de utilizar uma combinação de Python (Py), JavaScript (JS) e SQL para criar recursos e funcionalidades inovadoras na plataforma de streaming de música líder do mercado. No backend, utilizei Python em conjunto com frameworks como Django para desenvolver a lógica de negócios e integrar com bancos de dados SQL, como MySQL ou PostgreSQL. Além disso, utilizei SQL para escrever consultas otimizadas e gerenciar eficientemente os dados. No frontend, utilizei JavaScript e frameworks como ReactJS para criar interfaces responsivas e interativas, proporcionando aos usuários uma experiência musical excepcional. Trabalhar no Spotify foi uma experiência enriquecedora, onde pude aprimorar minhas habilidades em Python, JavaScript e SQL, e contribuir para o desenvolvimento de uma plataforma de renome global.

Tecnologias utilizadas:

No meu trabalho como desenvolvedor, tive a oportunidade de contribuir para projetos na renomada empresa Google. Durante esse período, utilizei uma variedade de tecnologias para criar soluções inovadoras. No desenvolvimento de aplicações web, utilizei linguagens como JavaScript (JS), HTML e CSS para criar interfaces modernas e interativas. Também trabalhei com frameworks populares, como Angular ou React, para otimizar o desenvolvimento e fornecer uma experiência de usuário excepcional. Além disso, utilizei tecnologias de backend, como Python ou Java, para criar lógica de negócios robusta e integrar com bancos de dados SQL ou NoSQL. Através do meu trabalho no Google, pude aprimorar minhas habilidades técnicas e participar de projetos de ponta que impactaram milhões de usuários em todo o mundo.

Foi uma experiência gratificante trabalhar na Twitch, onde pude aprimorar minhas habilidades técnicas e contribuir para o desenvolvimento de uma plataforma líder no setor de streaming de vídeo.
